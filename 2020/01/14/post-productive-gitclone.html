[Productive] GitHub Clone 할 때 폴더 째 혹은 내용물만 가져오기

Productive Git
Published on 14 Jan 2020

git clone 의 두가지 방법

레포지토리 폴더 통째로 가져오기

$>git clone repository

이렇게 하면 Repository이름의 폴더가 git clone을 실행한 Path로 복사된다.

레포지토리 내용물만 가져오기

$>git clone repository .

이렇게 하면 현재 Path로 Repository 의 내용물만 복사된다.
